## 💼 الملخص المهني

**مطور Fullstack** يتمتع بخبرة 4 سنوات في بناء تطبيقات الويب عالية الأداء. التخصص: **دورة التطوير الكاملة** — من تصميم واجهة المستخدم/تجربة المستخدم إلى النشر في الإنتاج. خبرة في المجموعة الحديثة (React, TypeScript, Python, Django, Docker)، إعداد خط أنابيب CI/CD، الاختبار الآلي، ونشر VPS مع اتباع أفضل ممارسات الأمان.

**🎯 الكفاءات الأساسية:**

- **Frontend:** React, TypeScript, Next.js, تصميم متجاوب، mobile-first, PWA
- **Backend:** Python (Django, Flask, FastAPI), Node.js, REST/GraphQL API
- **Desktop/Mobile:** Electron.js, React Native, Expo
- **DevOps:** Docker, CI/CD (GitHub Actions, GitLab CI), Linux, Nginx, SSL/TLS
- **Testing:** اختبار API، E2E (Playwright)، اختبار آلي
- **الأمان:** HTTPS, OAuth, JWT، نشر تطبيقات آمن

## 💼 الخبرة العملية والمشاريع

### 🛒 **منصة التجارة الإلكترونية** | مطور Fullstack | 2023

**Stack:** React, TypeScript, Next.js, Django REST, PostgreSQL, Docker, Nginx

- تطوير متجر إلكتروني كامل الميزات مع واجهة مستخدم متجاوبة
- تنفيذ نظام المصادقة (JWT)، عربة التسوق، تصفية المنتجات
- إعداد Django REST API مع التحكم في الوصول القائم على الدور (RBAC)، لوحة الإدارة
- النشر على VPS (Ubuntu + Nginx + SSL)، تكوين CI/CD عبر GitHub Actions
- **النتيجة:** تطبيق جاهز للإنتاج مع HTTPS آمن، نشر آلي

### 📋 **مدير المهام في الوقت الفعلي** | مطور Fullstack | 2023

**Stack:** React, FastAPI, WebSocket, PostgreSQL, Playwright

- إنشاء نظام إدارة المهام مع واجهة السحب والإفلات
- تنفيذ WebSocket للمزامنة في الوقت الفعلي بين العملاء
- كتابة اختبارات E2E (Playwright) مع تغطية أكثر من 80%
- تحسين الأداء: التحميل الكسول، مصفوفة القائمة الافتراضية
- **النتيجة:** تطبيق متجاوب يدعم 100+ مستخدم متزامن

### 🎓 **بوابة معلومات الكلية** | مطور Backend | 2022

**Stack:** Python, Django, Docker, GitLab CI/CD, Nginx, Ubuntu

- تطوير backend لنظام معلومات الكلية
- تكوين خط أنابيب CI/CD كامل (GitLab CI): الاختبارات → البناء → النشر
- أتمتة تجديد شهادة SSL (Let's Encrypt)، إعداد الوكيل العكسي
- إدارة VPS (Ubuntu)، تكوين المراقبة، التسجيل
- **النتيجة:** نظام إنتاج مستقر مع نشر آلي

### 🤖 **بوت إشعارات Telegram** | مطور Python | 2023

**Stack:** Python, aiogram 3.x, GitHub Actions, REST API

- إنشاء بوت Telegram للإشعارات الآلية وتكامل API الخارجي
- تنفيذ معالجة الأوامر، لوحات المفاتيح المضمنة، البرمجيات الوسيطة
- إعداد النشر الآلي عبر GitHub Actions
- **النتيجة:** البوت يعالج أكثر من 500 رسالة/يوم مع وقت تشغيل 99.9%

### 💻 **تطبيق سطح المكتب (Electron)** | مطور Desktop | 2023

**Stack:** Electron.js, React, TypeScript, SQLite

- تطوير تطبيق سطح مكتب متعدد المنصات (Windows/Mac/Linux)
- تنفيذ اتصال IPC، عمليات نظام الملفات، قاعدة البيانات المحلية
- إنشاء وظيفة التحديث التلقائي، مثبتات التطبيق
- **النتيجة:** تطبيق مع أكثر من 200 مستخدم نشط

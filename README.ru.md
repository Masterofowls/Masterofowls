## Обо мне

Fullstack-разработчик, DevOps-инженер и системный администратор с **6-летним** опытом программирования. Создаю и поддерживаю программные решения полного цикла — от интерактивных фронтендов и надёжных backend-сервисов до контейнеризированных развёртываний и серверной инфраструктуры. Одинаково уверенно работаю в **Windows** и **Linux**, оперируя на каждом уровне стека: написание production-кода, автоматизация CI/CD пайплайнов, управление базами данных, настройка веб-серверов и администрирование VPS.

| Роль                        | Специализация                                                              |
| :-------------------------- | :------------------------------------------------------------------------- |
| **Web-разработчик**         | React, Next.js, TypeScript — адаптивные SPA/SSR приложения                 |
| **Backend-инженер**         | Python (Django, FastAPI, Flask), Node.js — REST и GraphQL API              |
| **DevOps-инженер**          | Docker, CI/CD (GitHub Actions, GitLab CI), Nginx, SSL/TLS, мониторинг      |
| **Системный администратор** | Linux (Ubuntu/Debian) и Windows — управление серверами, SSH, systemd, cron |
| **Python-разработчик**      | Автоматизация, боты, обработка данных, скрипты, async-фреймворки           |
| **Desktop-разработчик**     | Tauri, Electron.js — кроссплатформенные нативные приложения                |

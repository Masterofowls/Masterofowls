## 💼 プロフェッショナル概要

4年の経験を持つ**フルスタック開発者**で、高性能なWebアプリケーションを構築しています。専門分野:**完全な開発サイクル** — UI/UXデザインから本番環境へのデプロイまで。モダンなスタック(React、TypeScript、Python、Django、Docker)、CI/CDパイプラインの構築、自動テスト、およびセキュリティベストプラクティスに従ったVPSデプロイに精通。

**🎯 コアコンピテンシー:**

- **フロントエンド:** React、TypeScript、Next.js、レスポンシブデザイン、モバイルファースト、PWA
- **バックエンド:** Python (Django、Flask、FastAPI)、Node.js、REST/GraphQL API
- **デスクトップ/モバイル:** Electron.js、React Native、Expo
- **DevOps:** Docker、CI/CD (GitHub Actions、GitLab CI)、Linux、Nginx、SSL/TLS
- **テスティング:** APIテスト、E2E (Playwright)、自動テスト
- **セキュリティ:** HTTPS、OAuth、JWT、安全なアプリケーションデプロイ

### 📋 **リアルタイムタスクマネージャー** | フルスタック開発者 | 2023

- 80%以上のカバレッジでE2Eテスト(Playwright)を作成

- **結果:** 100人以上の同時ユーザーをサポートするレスポンシブアプリケーション

## 💼 पेशेवर सारांश

4 साल के अनुभव के साथ **Fullstack Developer** उच्च-प्रदर्शन वेब एप्लिकेशन बनाने में। विशेषज्ञता: **पूर्ण विकास चक्र** — UI/UX डिज़ाइन से उत्पादन परिनियोजन तक। आधुनिक stack (React, TypeScript, Python, Django, Docker), CI/CD पाइपलाइन सेटअप, स्वचालित परीक्षण, और सुरक्षा सर्वोत्तम प्रथाओं के साथ VPS परिनियोजन में अनुभवी।

**🎯 मुख्य दक्षताएं:**

- **Frontend:** React, TypeScript, Next.js, उत्तरदायी डिज़ाइन, mobile-first, PWA
- **Backend:** Python (Django, Flask, FastAPI), Node.js, REST/GraphQL API
- **Desktop/Mobile:** Electron.js, React Native, Expo
- **DevOps:** Docker, CI/CD (GitHub Actions, GitLab CI), Linux, Nginx, SSL/TLS
- **Testing:** API परीक्षण, E2E (Playwright), स्वचालित परीक्षण
- **सुरक्षा:** HTTPS, OAuth, JWT, सुरक्षित एप्लिकेशन परिनियोजन

## How to Add Translations

To complete the multi-language support, each language file should include:

1. Language selector badges (already updated in EN and RU)
2. Translated contact information
3. Translated professional summary
4. Translated project descriptions
5. Translated career focus section

**Note:** Technical terms, technology names, and GitHub statistics remain the same across all languages.
